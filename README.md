# Sentiment Analysis Tool

A Python tool that analyzes the sentiment of text using NLP.

## Features
- Analyze single sentences interactively
- Bulk analyze CSV files
- Saves results to txt and CSV files
- Shows summary of positive, negative and neutral sentences

## Libraries Used
- TextBlob
- Pandas

## How to Run
1. Install libraries: pip install textblob pandas
2. For single sentences: python sentiment.py
3. For bulk analysis: python analyze_csv.py
